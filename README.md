# 🍔 Swiggy Sales Dashboard | Excel Project

## 📌 Project Overview
This project presents an interactive **Swiggy Sales Dashboard** built using **Excel** to analyze sales performance, customer ratings, and order trends across different states, cities, and food categories.

The dashboard helps businesses make data-driven decisions by visualizing key performance indicators (KPIs) and sales patterns.

---

## 🎯 Key Features

- 📊 Total Sales Analysis
- ⭐ Average Customer Rating
- 🛒 Total Orders Tracking
- 📅 Monthly, Weekly, and Daily Sales Trends
- 🍕 Sales by Food Type (Veg vs Non-Veg)
- 🗺️ State-wise Sales Distribution Map
- 🏙️ Top 5 Cities by Sales
- 📦 Category & Restaurant Filters (Slicers)
  

---

## 📈 Key Metrics (KPIs)

- **Total Sales:** ₹53.01 Million
- **Average Rating:** 4.34
- **Average Order Value:** ₹268.51
- **Rating Count:** 5.59 Million
- **Total Orders:** 197.43K

---

## 🛠️ Tools & Technologies Used

- Microsoft Excel
- Data Visualization
- Data Cleaning
- Data Modeling
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Details

The dataset includes:

- Order Date
- Sales Amount
- Rating
- State
- City
- Food Category
- Restaurant Name
- Order Quantity

No null values or duplicate records are present in the dataset.

---
## 📊 Insights from Dashboard

- Sales show consistent growth from February to August.
- Non-Veg food contributes **64%** of total sales.
- Karnataka has the highest sales among all states.
- Saturday records the highest daily sales.
- Average customer rating remains stable at **4.3**.
